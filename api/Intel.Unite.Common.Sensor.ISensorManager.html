Set(Sensor)

Sets a sensor to the list of all sensors. This overwrites all sensors that have the same: UniqueName, Id and ModuleId. If an exception is thrown, the values are not stored. If stored, raises the event SensorAdded. Note: It's responsibility of the owner of the sensor (ModuleId) to continually call Set to make sure it's sensor does not expires.
