Expiration

Gets or sets the Expiration date time of the sensor. We assume a sensor is continually updating it's values, thus properties are limited for how long they are valid before being purged. This must be set in UTC time. Maximum value is DateTime.UTCNow.AddHours(24). If value is greater than that, it will be limited to 24 hours.
